Time box: 1-2 hours

Purpose: attach types to the JavaScript you already know until type safety feels natural instead of heavy.

How to use this folder:
- Keep examples tiny.
- Read the type, then say it out loud.
- Predict errors before the editor does.
- Prefer understanding over memorizing syntax.
